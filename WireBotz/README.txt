===================================================================================
Essa é a biblioteca WireBotz que te ajudará a conectar vários Arduinos usando I²C

A documentação completa está disponível no drive da equipe

===================================================================================

INSTALANDO

Para usar a biblioteca:
	1. Encontrar o diretorio de bibliotecas da sua IDE do Arduino. No Linux
	   costuma ser ~/sketchbook/libraries .
	2. Copiar a pasta ./copy_to_libraries/WireBotz para o diretorio de bibliotecas.
	3. Após reiniciar a IDE do Arduino, o menu " Sketch > Import Library " deverá
	   conter o item WireBotz
	4. Para usar a biblioteca no seu código, é necessário usar os dois includes:
	      #include <Wire.h>
	      #include <WireBotz.h>
